- Install and update Nodejs
- Install Java jdk and runtime 17 -> sudo apt-get install oracle-java17-installer oracle-java17-set-default
    Execute 'java --version' if there is this output it's good:
        java 17 2021-09-14 LTS
        Java(TM) SE Runtime Environment (build 17+35-LTS-2724)
        Java HotSpot(TM) 64-Bit Server VM (build 17+35-LTS-2724, mixed mode, sharing)
    else execute 'sudo update-alternatives --config java' and choose java17 version.
- Link to install Minecraft Launcher if you don't have it -> https://www.launchershiginima.com/en/v4400/
- Link to download server.jar -> https://getbukkit.org/get/bf7ac3b5bc08ea97d22919680d240a80
    Launch server with 'java -Xms512M -Xmx1024M -jar spigot-1.18.1.jar' and accept change the variable to 'true' in eula.txt.
    When the server is already launch, stop it, go into server.properties file and change to 'false' the 'online-mode'.
- Install module and utilities ->
    npm i mineflayer --save
    npm i mineflayer-navigate --save
    npm i mineflayer-pathfinder --save
    npm i minecraft-data --save
    npm i vec3 --save
    npm install prismarine-viewer --save
- Link for github module https://github.com/PrismarineJS/mineflayer
